### Description

Image of oysters used for measurments, as well as the measurement images generated by ImageJ.

### FILES

- `plate-<A-Z>-USDA-<N>.jpg`: Images of 12-well plates with oysters and ruler.

     - `<A-Z>`: Letter corresponds to plate letter.

     - `<N>`: Number corresponds to the USDA family number.

- `*-ImageJ.jpg`: Corresponding image used by ImageJ for oyster area calculations.
