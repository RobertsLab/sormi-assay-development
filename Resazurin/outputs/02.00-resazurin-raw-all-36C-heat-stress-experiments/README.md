## Files

### `plot_df.csv`

Filtered, merged per-well fluorescence data across all freshwater
room-temperature resazurin assay experiments. Each row is one well read
at one timepoint. Blank wells and samples flagged `exclude_from_analysis`
in the source layout files are excluded.

| Column | Type | Description |
|--------|------|-------------|
| `row_id` | character | Plate row letter (e.g., `A`, `B`) |
| `col_id` | integer | Plate column number (e.g., `1`, `2`) |
| `well_id` | character | Normalized well identifier (e.g., `A1`, `B3`) |
| `value` | numeric | Raw fluorescence (RFU) measured at this timepoint |
| `plate_id` | character | Plate identifier within the experiment (e.g., `a`, `b`) |
| `time_hr` | numeric | Time elapsed since the start of the experiment (hours) |
| `exp_id` | character | Experiment date string (e.g., `20260429`) |
| `is_blank` | logical | `FALSE` for all rows (blank wells are excluded) |
| `exclude_from_analysis` | logical | `FALSE` for all rows (excluded samples are removed) |
| `family_id_group` | character | USDA oyster family identifier (e.g., `1`, `9b`) |
| `sample_id_group` | character | Within-experiment individual sample ID; **not unique across experiments** — use `unique_id` to track individuals |
| `unique_id` | character | Globally unique individual identifier constructed as `exp_id_sample_id_group` (e.g., `20260429_1`); use this to distinguish same-numbered animals from different experiments |
| `family` | factor | Factor version of `family_id_group`; levels are sorted alphanumerically |
| `experiment` | factor | Factor version of `exp_id`; levels are sorted chronologically |
